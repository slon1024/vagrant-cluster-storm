The storm cluster
=====================

Setup the  [storm] cluster (also [kafka], [zookeeper]) using [Vagrant] with [ansible].

## Run the first time
Run script `setup.sh` and follow according to tips.

## Run general
* start: `vagrant up`
* finish: `vagrant halt`
* other commands: `vagrant list-commands`
